# Tools Documentation

This section documents the Crusty development tools.

## Topics

- [crustydoc](crustydoc.md) - Documentation generator (rustdoc wrapper)
- [crustyfmt](crustyfmt.md) - Code formatting tool

## Overview

Crusty provides development tools to enhance the developer experience:

- **crustydoc**: Documentation generation using rustdoc
- **crustyfmt**: Automatic code formatting similar to rustfmt
